# WorkForce Management System

Java-based desktop application for managing employees and workforce data.

## Features
- Add Employee
- Update Employee
- Delete Employee
- View Employee Details
- Attendance Management
- Salary Management

## Tech Stack
- Java
- Swing
- MySQL
- JDBC

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Configure MySQL database
4. Run `Main.java`
